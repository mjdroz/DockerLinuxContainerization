About Page

This website is an assignment for a graduate level course, IS 601 - Web Systems Development. This course is offered at New Jersey Institute of Technology in Newark, New Jersey.

This assignment titled, "Website on Docker, Linux, and Containerization", is an information based assignment, which is going to help students gain a better understanding of Docker, Linux, and Containerization. The main goal behind this project is to help students understand what Docker is, how it works, and why we use it. The information learned here will be used in other projects throughout the semester.

Finally, any external sources will be linked at the bottom of each page for your information/research.
