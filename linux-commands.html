Basic Linux Commands

Linux Commands

Linux is an operating system that was created by Linus Torvalds in 1991. This operating system sees worldwide use, and has multiple distributions that millions of computers and servers run each day. Below are some basic commands, what they are, and how to use them.

Command Definition Code Format
pwd This command is used to print the path for your current directory. pwd
ls This command lists the contents of your current directory. ls
cd This command is used for navigation between directories. cd [directory name]
Change to the previous directory = "cd .."
Change to the root directory = "cd /"
cp This command is used to copy files. This can be used to copy files in the same directory or to a different directory. cp [filename] [new filename]
cp [filename] [directory to copy to]
rm This command is used to remove or delete files rm [filename]
mkdir This command is used to create a new directory. mkdir [directory name]
rmdir This command is used to remove an existing directory. Note: The directory that you plane to remove must be empty to use this command. rmdir [directory name]
rm -r This is an option for the "rm" command, talked about above. This should be used if you are trying to remove a directory that has content inside of it, since "rmdir" will not work in that instance. rm -r [directory name]
clear This command is used to clear the screen of all previous commands and outputs. It is good to do this once you finish a task so you are not confused. clear
man This command is used to check out the "man" pages. These pages are a guide to what command you type in. When in doubt, visit the man pages! man [command]
