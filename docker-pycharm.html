Connecting to Docker with PyCharm

Step 1:

Install Docker Engine (https://docs.docker.com/engine/install) (https://docs.docker.com/get-docker/) and make sure it is running and working properly. Install the PyCharm Pro IDE (https://www.jetbrains.com/pycharm/).

Step 2:

Once PyCharm is installed and open, you will need to create a few files in order to run a Python program using Docker. Those files include a Python file (which will be run) and a Dockerfile (which will be used to configure the container). After the files are created, you need to click on "Add Configuration" in the top right of the IDE to add a Docker configuration. From there click on the "+" button and then click on the "Docker" dropdown. Afterwards, click on "Dockerfile" and a configuration window should appear. Fill out the directory you want to configure, the name of the Docker image, and the name of the Docker container. After that make sure to check off the box that says "Run built image". When all of this is done, click "apply" and "ok".

Setting up the Docker configuration in PyCharm.

Step 3:

Part 1:

Click on the "services" tab at the bottom of the screen to make sure that Docker is pair correctly (shown below). Also make sure that it says "Dockerfile" where "add configuration" was (shown below).

Setting up the Docker configuration in PyCharm.

Part 2:

Click on the interpreters section on the bottom right of the screen. Then click "add an interpreter" and a popup menu will appear (shown below). Click on the "Docker" button and then click "ok". This may take a moment to configure.

Setting up the Docker configuration in PyCharm.

Step 4:

Finally, click the play button in the top right corner of the screen to run the program. You should see a "build log" of the container gathering the necessary programs and libraries to run the Python code. Once this is done, the container will then build the Docker image and run the program. To see the output of the program, click on the "Attached Console" tab next to the "Log" tab (shown below).
